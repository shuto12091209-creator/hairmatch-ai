# HairMatch AI 要領書

## 1. アプリ概要

HairMatch AIは、ユーザーがアップロードした顔写真をもとに顔型を判定し、おすすめの髪型を表示するWebアプリです。

顔写真からMediaPipeを使用して顔の特徴点を取得し、顔の縦横比を計算することで、以下の顔型を判定します。

* 面長
* 卵型
* 丸顔

自分に似合う髪型が分からないという課題に対して、顔型をもとに髪型を提案することを目的としています。

## 3. 処理の流れ

```text
upload.phpで画像をアップロード
↓
result.phpで画像を一時保存
↓
Docker内のPythonを実行
↓
MediaPipeで顔の特徴点を取得
↓
顔の縦幅・横幅を計算
↓
顔型を判定
↓
JSON形式でPHPに結果を返す
↓
result.phpで診断結果を表示
↓
ページ離脱時に画像を自動削除
```

---

## 4. 画像アップロード処理

`upload.php` では、ユーザーが顔写真を選択し、`result.php` にPOST送信します。

画像ファイルを送信するため、formタグには以下の設定を使用しています。

```html
<form action="result.php" method="POST" enctype="multipart/form-data">
```

`enctype="multipart/form-data"` を指定することで、PHP側で画像ファイルを受け取れるようにしています。

---

## 5. result.phpの役割

`result.php` では、以下の処理を行います。

1. アップロード画像を受け取る
2. 画像を `uploads` フォルダに一時保存する
3. Docker内のPythonスクリプトを実行する
4. Pythonから返ってきたJSONを読み取る
5. 顔型とおすすめ髪型を画面に表示する
6. ページ離脱時に画像削除処理を呼び出す

PHPからPythonを実行することで、Web画面から画像解析処理を利用できるようにしています。

---

## 6. Pythonによる顔型判定

Python側では `detect_face.py` を使用しています。

主な処理は以下です。

1. 画像をOpenCVで読み込む
2. MediaPipeで顔のランドマークを取得する
3. 額・顎・左右の頬の特徴点を取得する
4. MediaPipeの座標をピクセル座標に変換する
5. 顔の縦幅と横幅を計算する
6. 縦横比から顔型を判定する
7. 判定結果をJSON形式で出力する

---

## 7. 顔型判定ロジック

現在は、顔の縦横比をもとに顔型を判定しています。

```python
face_ratio = face_height / face_width
```

判定条件は以下です。

```text
1.25以上 → 面長
1.05以下 → 丸顔
それ以外 → 卵型
```

MediaPipeの座標は0〜1の割合で取得されるため、そのまま計算すると画像サイズの影響を受ける可能性があります。
そのため、画像の横幅・縦幅を使ってピクセル座標に変換してから、顔の縦幅・横幅を計算しています。

---

## 8. JSON出力

Pythonは判定結果をJSON形式でPHPに返します。

成功時の例：

```json
{
  "success": true,
  "face_shape": "面長",
  "face_ratio": 1.31,
  "face_height": 512.3,
  "face_width": 389.4,
  "recommendations": ["センターパート", "アップバング", "ウルフ"]
}
```

失敗時の例：

```json
{
  "success": false,
  "message": "顔を検出できませんでした"
}
```

JSON形式で返すことで、PHP側で `face_shape` や `recommendations` を取り出し、画面表示に利用できるようにしています。

---

## 9. 画像削除処理

アップロードされた画像は、診断処理のために一時的に保存されます。

診断結果ページを離れたタイミングで、JavaScriptの `sendBeacon()` を使用して `delete_image.php` に削除依頼を送信します。

```javascript
navigator.sendBeacon("delete_image.php", data);
```

これにより、不要な顔写真をサーバー上に残さない設計にしています。

`delete_image.php` では、送信された画像ファイル名をもとに、`uploads` フォルダ内の該当ファイルを削除します。

また、セキュリティ対策として、`basename()` や `realpath()` を使用し、`uploads` フォルダ外のファイルを削除できないようにしています。

## 11. 現在の課題

現在は顔の縦横比を中心に判定しているため、境界に近い顔型では誤判定が発生する場合があります。

特に以下のようなケースで誤判定が起こる可能性があります。

* 丸顔と卵型の境界
* 面長と卵型の境界
* 顔が傾いている画像
* 髪型で輪郭が隠れている画像
* 正面画像ではない画像
* 証明写真風ではない画像

---

## 12. 今後の改善予定

今後は、顔の縦横比以外の特徴量も追加し、判定精度を向上させる予定です。

追加を検討している特徴量は以下です。

* エラの張り具合
* 額の広さ
* 顎先の細さ
* 頬骨の幅
* ベース型・逆三角形への対応
* テスト画像数の増加
* 判定ロジックの改善

現在はMVPとして3分類に対応していますが、今後はより細かい顔型分類にも対応できるよう改善していく予定です。

---

## 13. アピールポイント

このアプリでは、単に画像をアップロードするだけでなく、PHPとPythonを連携させて画像解析を行っています。

また、MediaPipeで取得した顔ランドマークをもとに、顔の縦横比を計算し、顔型を判定する処理を実装しています。

さらに、アップロードされた顔写真は診断結果ページを離れたタイミングで自動削除されるため、不要な個人画像をサーバー上に保持しない設計にしています。

実装だけでなく、テスト画像を用いた精度検証も行い、判定結果を数値として確認しています。

このように、機能実装・画像解析・PHPとPythonの連携・Docker環境構築・プライバシー配慮・精度検証まで一通り行っている点が特徴です。
